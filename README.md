# LifeOS Full Document Pack

This pack contains the current LifeOS architecture/planning documents.

## Contents

```text
docs/
  LIFEOS_HANDOFF.md
  PLATFORM_ARCHITECTURE.md
  MOBILE_PLAN.md
  WEBSITE_PLAN.md
  README_UPDATE_SECTION.md
  PHASE4_INSTALL.md
  PHASE5_INSTALL.md
```

## How to use

1. Copy `PLATFORM_ARCHITECTURE.md`, `MOBILE_PLAN.md`, and `WEBSITE_PLAN.md` into your repo's `docs/` folder.
2. Copy the section from `README_UPDATE_SECTION.md` into your main `README.md`.
3. Keep `LIFEOS_HANDOFF.md` as the current continuation/handoff summary.
4. Use `PHASE4_INSTALL.md` and `PHASE5_INSTALL.md` as implementation references.

## Recommended commit

```powershell
git add docs README.md
git commit -m "Document LifeOS platform architecture and plans"
```
