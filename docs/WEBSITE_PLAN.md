# LifeOS Website Plan

The LifeOS website is the public showroom for the project.

It should explain:

- What LifeOS is
- The weekly pressure command centre idea
- Desktop vs mobile model
- TimerAgent as the first shipped desktop utility
- Screenshots
- Roadmap
- Release notes
- GitHub links

The website is not the app.

The first website should be static and informational. No login portal, cloud dashboard, payment system, or public SaaS account system yet.
