# v5 Handoff Checklist

- Connector scopes documented.
- Secrets excluded from repository and logs.
- Token storage strategy reviewed.
- User can disconnect the connector.
- Preview provenance is complete.
- Duplicate keys are deterministic.
- Refresh is visible and user-controlled.
- Errors are surfaced without data loss.
- Raw records remain read-only.
- Acceptance does not equal target mutation.
- Target-module validation remains authoritative.
- Audit trail is readable.
- Demo and real data are visibly separated.
- Expected/imported money remains unsafe.
- Rollback and rejection paths are tested.
