# Group 40 public content source map

| Public state | Pack 1 source | Validation boundary |
|---|---|---|
| Product status | `PublicContent.Products` seeded from the approved Group 40 baseline | One model used across Website pages |
| Documentation index | `PublicContent.Docs` | Required metadata and deterministic local search tests |
| Required routes | `PublicContent.Routes` and individual Razor route declarations | Route inventory test plus manual navigation |
| Privacy updated date | `PublicContent.LastUpdated` | Rendered by Privacy page |
| Test totals | `PublicContent.ProofMetrics` | Preserves separate Desktop/Core and Companion baselines |
| Public proof | Sanitized first-party Website copy | No private repository documents copied into static output |

A later group may introduce build-time generation from repository status files. Group 40 deliberately avoids publishing private repository documentation or creating a broad extraction pipeline.
