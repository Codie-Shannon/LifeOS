# Mobile Companion v0.1.0-beta.1 — Group 34

This checkpoint hardens the intentionally lightweight Companion. It remains a separate application from the future full Mobile app.

Implemented scope: compact read-only glance cache; explicit refresh; stale/offline wording; opt-in private notifications; stale Sending recovery; no reconnect auto-send; explicit retry; duplicate versus changed-content conflict distinction; explicit conflict choices and audit persistence; pairing reset separated from local data; local-data summary; accessibility-oriented labels and plain-language recovery; beta readiness view.

Excluded: cloud relay, accounts, background sender, periodic sync, automatic retry, app-closed execution, broad attachment/photo workflows and full Mobile modules.
