## What changed

v1.6 adds the first dedicated Daily Operating Flow layer.

It introduces local-first blocks for:

- anchors
- next actions
- waiting checkpoints
- admin sweeps
- proof capture
- low-energy fallback blocks
- recovery blocks
- stop points

## Why this exists

Daily Operating Flow is not a task manager. It controls the day so LifeOS can show what should move now, what should wait, what should be parked, and where the stop point is.
