# LifeOS v1.8 Code Release Notes

## Version

**Version:** v1.8 code checkpoint  
**Screenshot group:** Group 04 — Settings / Safety / Theme

## What changed

v1.8 adds a local Settings / Safety / Theme foundation.

It adds:

- local settings/safety/theme profile model
- strict/balanced/experimental-local-only safety modes
- local-only mode guardrail
- manual-review-before-send guardrail
- expected-money-is-not-safe guardrail
- screenshot privacy guardrail
- destructive-action confirmation preference
- demo-safe data preference
- appearance and accent preferences
- Settings / Safety desktop workspace
- Command Centre safety and guardrail signals

## Operating rule

LifeOS should stay conservative before integrations. Local-only mode, manual review, screenshot privacy, and expected-money safety remain visible as first-class settings.

## Boundary

v1.8 does not add account management, permissions, encryption, live theme switching, cloud sync, integrations, or automatic messaging.
