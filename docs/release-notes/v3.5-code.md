## What this checkpoint adds

v3.5-code adds the Search / Knowledge Centre foundation.

It creates local knowledge items, source boundaries, search profiles, manual review state, and future integration landing structure.

## Boundary

v3.5 is not external search, full-text indexing, connector search, inbox scanning, cloud sync, AI reasoning, automatic summarisation, or final knowledge automation.

It is the local search/knowledge shape that future integrations and AI assistant layers can feed.
