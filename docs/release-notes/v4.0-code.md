## What v4.0 adds

v4.0 starts the LifeOS spine completion road.

It restores the older/canon LifeOS operating spine as the authority before integrations:

- Command Centre
- Pressure Engine
- Money Pressure
- Safe-to-Spend
- Money Timeline
- Bills / Subscriptions
- Upcoming Payments
- Pay Later / Zip / Afterpay
- Money Profile / Hidden Deductions
- Agenda
- Payment Calendar
- Follow-Ups / Waiting-On
- Relationship / Email Radar
- Work Pipeline / Opportunity Tracker
- Work Sessions
- Paid Work Centre
- Proof Tracker
- Evidence Vault
- Receipt OCR / Evidence-to-Item
- Weekly Close-Out
- Integration Inbox / Sync Preview Queue
- Universal Spine
- Search / Knowledge
- Settings / Safety

## Master rule

Everything important becomes an item. Every item has state. Every state affects pressure. Every pressure feeds the Command Centre.

## Boundary

v4.0 maps the LifeOS spine.

It is not real OAuth, live email/calendar/accounting/bank sync, AI assistant execution, the companion app, or the major workspace redesign.

Real integrations move to v5. The companion app checkpoint is v6.5.
