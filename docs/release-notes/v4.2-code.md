## What v4.2 adds

v4.2 adds the Bills / Payments engine.

It models:

- bills
- subscriptions
- upcoming payments
- Pay Later / Zip / Afterpay instalments
- Pay Later remaining balances
- hidden deductions
- tax/GST/ACC/student-loan/KiwiSaver/custom reserve placeholders
- manual cashflow items
- review/evidence gates
- safe-to-spend drag
- due pressure
- local JSON storage
- Command Centre v4.2 money-obligation signals

## Boundary

v4.2 is local/manual money-state modelling.

It does not connect bank feeds, Pay Later providers, accounting systems, email, OCR automation, OAuth, or real external sync.
