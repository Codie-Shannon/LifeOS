## What v4.3 adds

v4.3 adds the Money Profile workspace.

It models:

- confirmed cash
- visible balance
- expected money excluded from safe money
- weekly known obligations
- hidden deductions
- tax/GST/ACC/student-loan/KiwiSaver/custom reserve placeholders
- minimum buffer
- emergency hold
- safe-to-spend before hidden deductions
- safe-to-spend after hidden deductions
- final safe-to-spend
- safe-to-spend confidence gates
- local money profile JSON storage
- Command Centre v4.3 money profile signals

## Boundary

v4.3 is local/manual safe-to-spend modelling.

It does not connect bank feeds, payslips, IRD/tax portals, accounting systems, open banking, email, OCR automation, OAuth, or AI actions.
