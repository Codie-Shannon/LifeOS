## What v4.5 adds

v4.5 strengthens the existing Work Pipeline instead of creating a duplicate tracker.

It adds:

- v4.5 Work Pipeline operating calculator
- review buckets
- money state classification
- operating signals
- operating summary
- shared bridge notes for item/state, Payment Calendar, and proof/paid-work lanes
- upgraded Work Pipeline workspace
- Command Centre v4.5 Work Pipeline cards and panels
- pipeline integrity warnings
- local/manual v4.5 boundary documentation

## Why this matters

Work Pipeline is the operating lane for:

- active work
- warm leads
- blocked projects
- waiting-on states
- follow-ups
- invoice readiness
- timesheet readiness
- payment expected state
- proof/evidence gaps
- parked opportunities

Expected pipeline value is visible for planning but remains excluded from safe money until paid.

## Boundary

v4.5 does not:

- send messages
- sync email/calendar/accounting/bank data
- create invoices
- create calendar events
- run AI actions
- replace a CRM
- perform the major workspace redesign
