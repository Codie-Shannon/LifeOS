# LifeOS v5.0.0-alpha.5 — Group 25

Group 25 adds one private/testing-mode authenticated Gmail connector using only `gmail.readonly`. Searches are explicit, profile-bound, date-bounded to 31 days, capped at 100 results, previewed before retrieval, and manually confirmed. Gmail messages normalize into the existing provider-neutral Email Radar evidence model and remain untrusted until reviewed.

No sending, drafts, replies, forwarding, archive, trash, delete, labels, stars, read-state mutation, attachment download, active HTML, background polling, scheduled scanning, Outlook, IMAP, POP3, AI interpretation, automatic Follow-Up, or Work Pipeline mutation is included.
