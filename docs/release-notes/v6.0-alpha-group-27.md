# LifeOS v6.0.0-alpha.1 — Group 27

## Controlled automation foundation

This release introduces visible automation intent without unattended execution. Rules are disabled by default, evaluation is manual and deterministic, inputs must be reviewed or trusted, and every matched result becomes a proposal requiring explicit review.

Approval records the user's decision but does not execute an operational change. High-risk external, communication, calendar, financial, destructive and script capabilities are blocked by policy.

## Known limitations

Manual evaluation only; dry-run only; no background worker, scheduler, timers, startup rules, automatic retries, external writes, email/calendar mutation, automatic Follow-Up, automatic Work Pipeline update, financial mutation, destructive action, scripts, AI, Companion App or major UI redesign.
