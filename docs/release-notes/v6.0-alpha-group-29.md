# LifeOS v6.0.0-alpha.3 — Controlled orchestration and recovery

Group 29 extends guarded internal automation into local, review-only orchestration.

- Schedules expose review intent; they never execute work.
- Supported schedule definitions: Manual only, One-time review date, Weekly review day.
- Due occurrences are singular and enter a visible Due Work queue.
- Explicit Start creates a persisted run paused before its first step.
- Every step requires an exact before/after preview and explicit confirmation.
- Only one typed Low-risk reversible internal step executes at a time.
- Runs pause between steps and after failures.
- Recovery is explicit: Retry, optional Skip, Cancel remaining, or run-scoped reverse-order rollback.
- In-progress execution never resumes automatically after restart.
- Email, mailbox, calendar, finance, destructive actions, scripts, processes, plugins and AI remain blocked.
