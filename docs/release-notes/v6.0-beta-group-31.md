# LifeOS v6.0.0-beta.1 — Group 31

Group 31 is the v6 controlled automation release checkpoint.

## Added
- Unified persisted-state Automation Readiness view.
- Explicit v6 automation store schema and safe migration normalization.

## Preserved boundaries
- Approval never executes.
- Orchestration remains one-step-at-a-time.
- Emergency Stop is persisted and distinct from the normal pause.
- No unattended runtime, automatic retry, external writes, financial/destructive mutation, scripts, processes, plugins or AI decisions.
