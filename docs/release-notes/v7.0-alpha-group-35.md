# LifeOS v7.0.0-alpha.1 — Group 35

Group 35 introduces the first source-backed Assistant workspace as a separate read-only reasoning surface above trusted local LifeOS state.

## Boundary

The Assistant can answer, summarise, explain and provide plain non-executing suggestions. It cannot execute, approve, confirm, continue orchestration, mutate state, write to connectors, run scripts/processes/plugins, search the web or create durable memory.

## Foundation

- Provider-neutral answer-provider abstraction.
- Approved source registry with persisted per-source enablement.
- Bounded question and retrieval limits.
- Source record IDs, timestamps and provenance preserved.
- Fact, inference, missing data and uncertainty classified separately.
- Explicit insufficient-evidence and mutation refusal responses.
- Session-only answer history.

Group 36, full Mobile, Website, autonomous planning and external AI actions remain out of scope.
