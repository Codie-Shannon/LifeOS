# LifeOS Desktop v7.0.0-alpha.3 — Group 37

## Scope

Group 37 introduces reusable review-only planning blocks and a controlled handoff into existing review surfaces.

## Added

- Stable review-only plan and block identities.
- Goal, Evidence, Constraint, Step, Dependency, Risk, Decision, Verification and Handoff blocks.
- Claim-to-source provenance, explicit assumptions, conflicts, stale evidence and missing-data gaps.
- Proposed, NeedsInput, Blocked and ReadyForReview states.
- Edit, reorder and remove controls before handoff.
- Exact payload preview, explicit review target and human confirmation.
- Session-local review artifacts with `Executable: No`.

## Safety

No operational state mutation, execution handler, durable memory, tool path, autonomous planning, external web search or connector write was introduced. Cancellation creates no partial review or operational state.
