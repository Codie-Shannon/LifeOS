# LifeOS Desktop v7.0.0-alpha.4 — Group 38

Group 38 adds explicit, user-controlled durable Assistant context. The new Memory workspace supports exact preview, type, scope, sensitivity, expiry, provenance, auditable edits, revocation and deletion. Relevant permitted memory is disclosed separately from direct evidence.

Safety remains strict: no automatic retention, hidden profiling, secret storage, external memory service, autonomous tools, background planning, plan conversion or execution. Current trusted LifeOS records always outrank conflicting remembered context.
