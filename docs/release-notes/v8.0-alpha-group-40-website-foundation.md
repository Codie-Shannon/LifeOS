# LifeOS v8 — Group 40 Website foundation (pre-screenshot)

Target Website release: **v0.1.0-alpha.1**

## Delivered in Pack 1

- .NET 10 standalone Blazor WebAssembly Website.
- Small public navigation with Product, Solutions, Docs, Evidence, Roadmap and Access.
- Required product, audience, Safety, Docs, Evidence, Roadmap, Access, About and Privacy routes.
- Browser-driven light/dark theme with a persisted user override.
- Token-based accent and status system.
- Responsive global header, mobile navigation and footer.
- Searchable local documentation index with structured metadata.
- Honest disabled Early Access preview with no collection.
- Static-first Cloudflare Pages publish configuration.
- Website tests for routes, product boundaries, content metadata, search and privacy/tracking boundaries.

## Explicitly excluded

Screenshots, downloads, authentication, accounts, portal APIs, billing, licensing, analytics, tracking, mass documentation migration, Desktop redesign and Full Mobile.

This is a pre-screenshot release note. Group 40 is not complete until Pack 2 commits exactly eight approved screenshots and final validation evidence.
