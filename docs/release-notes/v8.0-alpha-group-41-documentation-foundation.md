# LifeOS v8 Group 41 - documentation foundation

Website release: `v0.2.0-alpha.1`

Group 41 publishes the first controlled public LifeOS documentation set. It adds a repository-backed inventory, structured metadata, stable routes, deterministic filters, breadcrumbs, related documents and public evidence links.

A selected Desktop Assistant surface now keeps concise action-level safety guidance and offers an allowlisted `Open Documentation` link. Desktop remains usable offline and the full public document is not required for immediate safety.

Public output remains static-first, contains fictional or sanitized examples only and is checked for private paths, tracking scripts and forbidden content patterns.
