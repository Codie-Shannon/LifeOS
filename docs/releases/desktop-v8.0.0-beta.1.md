# LifeOS Desktop v8.0.0-beta.1

Status: beta release checkpoint

LifeOS Desktop v8 closes the three-group shell, workspace and operating-experience migration.

## Included

- Eight permanent core workspaces: Home, Work, Career, Money, Life, Projects, Assistant and Settings.
- Career remains separate from client Work.
- Balanced Home overview with Home-first or Last Used startup preference.
- Permanent top bar with current workspace, Ctrl+K command, Review, system/sync status, Emergency Stop, context and profile.
- Persistent contextual panel with automatic relevance, keyboard focus return and narrow-window collapse.
- Final Settings sections for appearance, navigation, accessibility, profiles, privacy/local data, sync/companion, automation/stop, integrations placeholder and diagnostics.
- Light, Dark, System and High Contrast themes.
- Purple, Blue and Teal accents.
- Comfortable and Compact density, Reduced Motion and approved text scaling.
- Safe preference normalization and corrupt-state fallback.
- Final allowlisted module route map with obsolete shell navigation removed.
- Deterministic Group 45 evidence contract and validation tooling.

## Boundary

This checkpoint does not add Microsoft or Google integrations, Full Mobile, money expansion, Career Studio, Grocery Planner, guarded writes or DevOps. Existing operational records remain in their authoritative stores and are not copied into shell preferences.
