# v4.5 — Work Pipeline

## Summary

v4.5 strengthens Work Pipeline into the operating lane for active work, warm leads, blocked work, follow-ups, invoice readiness, payment expected state, and proof gaps.

## Added / proven

- Work Pipeline dashboard metrics.
- Pipeline open, active, waiting, blocked, follow-up, timesheet, invoice, proof, expected value, and parked-opportunity views.
- Work Pipeline rule and operating signals.
- Today triage, active work lane, waiting-on-me, waiting-on-others, blocked work, invoice/timesheet readiness, expected payment not-safe-money lane, proof/evidence gaps, warm/parked opportunities, spine bridge, stage breakdown, filters, add form, and item list.
- Command Centre now exposes v4.5 Work lane metrics.

## Boundary

v4.5 does not send messages, sync external systems, create invoices, create calendar events, run AI actions, or replace a CRM. It models local state and pressure only.
