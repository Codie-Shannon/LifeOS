## Verification notes

The screenshots show:

- the app shell remains intact
- sidebar shows Desktop v4.2
- Command Centre header shows v4.2
- the Bills / Payments nav entry is visible
- Command Centre includes v4.2 money-obligation signals
- Bills / Payments workspace is reachable from the sidebar
- bills, subscriptions, Pay Later / BNPL, upcoming payments, hidden deductions, and imported receipt candidates are separated
- due pressure and safe-to-spend impact are visible
- Pay Later / Zip / Afterpay remaining balance and instalment load are visible
- hidden deductions are visible before spending
- untrusted or evidence-needed money items stay in review
- paid state requires confirmed evidence
- v4.2 boundary states no real bank feeds, Pay Later provider sync, accounting sync, email sync, OCR automation, or OAuth yet
- v4.3 is confirmed as the next lane: Money Profile / Hidden Deductions / Safe-to-Spend
