# Group 16 - Receipt OCR / Evidence-to-Item

LifeOS Desktop v4.6 turns local receipt or OCR output into reviewable evidence candidates before they can become trusted LifeOS items.

## What this group proves

- OCR or imported receipt data remains untrusted until manually reviewed.
- Source evidence is required before a receipt can become trusted.
- Accepted receipts remain separate from raw OCR candidates.
- Receipt value is visible without being treated as safe money.
- Receipt evidence can support bills, money review, paid work, and proof.
- The current implementation is local and manual only.
